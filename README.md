# Valentine 2026

A React website with a soft, romantic homepage: cream background and blurred pink hearts.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/`.
